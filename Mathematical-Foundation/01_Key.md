Single Key: Single Key is only used one time for a single message, each time a new key is generated for each message.
E.g: Encrypted Email: new key is generated for every email.

Multi Key: Single key is used to encrypt multi messages
E.g: Encrypted Files: Same key is used to encrypt multiple files

Cryptography is not the solution to every security problem like software bugs , social engineering.

Anything that can be done wwith the trusted authority can also be done without it.

1. Unforgeable.
2. Propose a construction
3. Anyone want to break the construction need to solve hard problem.
